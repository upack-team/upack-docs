title: Users Started
---

Upon logging into the system, you can access the "Process" module, where you can initiate, approve, and inquire about application forms.

 ![](https://console.steedos.cn/api/files/images/ZLgzmETofKEgsEL6y)


You can get an overview of the application form's progress by checking the "Process Diagram" located at the top of the page. The highlighted node on the process diagram indicates the current step of the application form.

 ![流程图](https://console.steedos.cn/api/files/images/W227Nw6yTwuWGAA7k)
