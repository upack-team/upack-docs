# 创建一个 Steedos DX 项目

设置项目的最简单方法是使用 create-steedos-app。这个命令行工具允许您快速开始构建一个新的 steedos 项目，并为您设置好所有东西。

## 创建一个空项目

```shell
npx create-steedos-app my-project
```

## 导入模板项目

您可以使用 -e 或 --example 参数来导入模板项目。

例如：https://github.com/steedos/steedos-examples

```shell
npx create-steedos-app my-project -e https://github.com/steedos/steedos-examples/tree/main/with-amis
```
